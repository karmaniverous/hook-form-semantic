      Map the RHF UI schedule (UI-friendly types) to the rrstack engine schedule.

      • 'span' → undefined freq
      • Date | null → epoch ms | undefined
      • tolerant CSV text → number[]
      • arrays pass through unchanged Assumes timezone has been validated upstream.

    Components

    React Hook Form field components built on Semantic UI React. Each guide shows a focused, real-world usage snippet inside a react-hook-form context.

    • HookFormField - Generic RHF + Semantic UI wrapper for any input.
    • HookFormCheckbox - Controlled checkbox with optional control labels.
    • HookFormDatePicker - Single date/time with Include Time toggle.
    • HookFormDateRangePicker - Date or datetime ranges with presets.
    • HookFormJsonEditor - JSON editor (vanilla-jsoneditor) integration.
    • HookFormMenu - Semantic UI Menu as a selector.
    • HookFormMenuDisplayMode - Opinionated Cards/Table selector.
    • HookFormNumeric - Number input via react-number-format.
    • HookFormPhone - International phone input with validation.
    • HookFormRRStack - Rich scheduling/rules editor (RRStack).
    • HookFormSort - Pair of dropdown + direction button for sorting.
    • HookFormWysiwygEditor - Rich text editor returning HTML.

    Notes

    • Examples are intentionally small and focused. They illustrate common real-world patterns without any extra fluff.
    • Import only the pieces you need in your app (see README “Install peers per component”).
